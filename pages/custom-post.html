Configuración del custom post

  1. Instalar plugin Custom Post types UI y Advanced custom fields
  2. Crear el post type en CPTUI (has archive true, hierarchical true, elige menu icon)
  3. Añade las taxonomias asignadas al post type creado(hierarchical true)
  4. Crear en custom fields el field dabel (relleno, instrucciones, valor requerido, lógica condicional...)
  5. Crear los custom post correspondientes para visualizar
